# God Vessel GDD v0.2

## タイトル（仮）

**Project: God Vessel**

---

## 1. ゲーム概要

### ジャンル
3D Co-op Action Roguelite Hack & Slash

### 一言で説明
プレイヤーは“神の器”として崩壊した異世界を探索し、武器融合と属性反応で独自ビルドを構築しながら、神々の戦争を生き抜く。

### コア体験
- 壊れビルド完成の快感
- 協力探索
- 強力な武器収集
- 武器融合による試行錯誤
- 属性反応コンボ
- 神との戦闘
- 異世界への没入感

---

## 3. 世界観

### 背景
神々はそれぞれ世界を創造したが、“誰が真の頂点か”を決めるため神同士の代理戦争を開始。プレイヤーは神性を宿した“器”として異形の存在が支配する世界へ送り込まれる。

### 世界の特徴
- ダークファンタジー
- 異世界感
- 不気味な存在
- 神性汚染
- 崩壊した文明

## 6. 属性システム

### 属性一覧（仮）
- 炎
- 氷
- 雷
- 毒
- 闇
- 光
- 風

### 属性反応（例）
- 炎 + 風 = 火炎竜巻
- 氷 + 雷 = 凍結麻痺
- 毒 + 炎 = 毒霧爆発
- 闇 + 重力 = 吸引
- 光 + 雷 = 連鎖放電

## 17. Non-Goals（MVPでは実装しない）

### 目的
スコープ肥大化を防ぎ、コア体験（戦闘・ビルド・融合）に集中する。

### MVP除外項目
- オープンワールド
- 無限ランダム生成
- 大量職業
- 複雑な分岐ストーリー
- PvP
- クラフト中心化
- 建築
- 巨大スキルツリー
- 武器種大量追加
- リアルグラフィック路線
- 巨大ボス大量実装
- MMO化
- ゲーム内AI生成

### 代替方針
- 出撃型セッション制
- 固定マップ + 可変要素
- 1職業 + 1武器種（初期）
- Lore断片による世界観表現
- Co-op特化
- 分解/強化/融合に限定
- 小規模スキルノード

### 優先度
- **S**: 戦闘気持ちよさ / 武器ドロップ / ビルド成立 / 融合 / 属性反応 / Co-op
- **A**: ボス / 世界観 / 拠点 / 成長
- **B**: 職業追加 / 武器種追加 / マップ追加
- **C**: ストーリー拡張 / PvP / 建築 / 大規模化
